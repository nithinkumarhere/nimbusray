Back End Developer & App Lead

Role Description
We are looking for a Backend developer responsible for development of backend services for mobile applications and devices in our new IoT product ecosystem.

You will be a critical member of a small team working to provide backend services for various components of our new IoT product ecosystem from design to production. Your primary focus will be database design and development of services to be consumed by mobile applications and devices.

Working throughout the project delivery lifecycle, you will successfully collaborate with multiple stakeholders and colleagues across multiple specialisms which could include; end users, key client representatives, business analysts, mobile app developers, graphic designers, operations teams and more. You will be taking up ownership of the development of the web backend, working with the Mobile Development and Data Analytics Team, developing and rolling out releases as planned.

    Key skills (3-4 years exp)

  • Experience in Java with strong server side technologies.
  • Relational Database design and SQL working knowledge
  • Strong grasp on concepts related to OOPs, API's and libraries.
  • Excellent analytical and problem solving skills.
  • At least 3 years of development experience.
  • Understanding fundamental design principles behind a scalable application
  • You know what it takes to design good REST-ful APIs
  • Proficient with code versioning tools such as Git or Subversion

    Desired

  • Experience with implementation of Data Analytics algorithms
  • Performance optimization, SQL tuning, caching techniques is a plus.
  • Proficiency in front-end technologies, such as HTML5, and CSS3
  • Hands-on experience with implementing Front-end application using JavaScript (jQuery / Angular / ReactJS / Backbone), HTML5, CSS3 technologies required
  • Ability to lead app development work
